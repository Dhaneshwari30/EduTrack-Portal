# EduTrack Portal

## Overview
EduTrack Portal is a Classroom and Student Management System developed using Spring Boot and MongoDB.

## Features
- Batch Management
- Student Management
- CRUD Operations
- REST APIs
- MongoDB Integration

## Tech Stack
- Java
- Spring Boot
- MongoDB
- Maven

## Project Structure

```text
src/
pom.xml
docs/
```

## Run Project

1. Start MongoDB
2. Configure application.properties
3. Run

```bash
mvn spring-boot:run
```
